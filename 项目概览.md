# 白板前端 (baibanfront) 项目概览

## 项目基本信息

- **项目名称**: @yunke/frontend
- **版本**: 0.21.0
- **授权**: MIT
- **作者**: toeverything
- **项目描述**: 白板前端，基于 Affine 二次开发

## 项目架构概述

该项目是一个基于 Affine 的白板应用前端项目，采用了模块化的 monorepo 架构。项目主要分为以下几个核心模块：

### 主要技术栈

- **构建工具**: Yarn 4.9.1 (workspaces)
- **TypeScript**: 5.7.2
- **打包工具**: Vite 6.0.3
- **测试框架**: Vitest 3.1.3
- **代码质量**: ESLint, Prettier, OxLint
- **Git Hook**: Husky + lint-staged

### 核心模块结构

```
baibanfront/
├── blocksuite/              # 核心块套件模块
│   ├── affine/             # Affine 相关功能
│   ├── framework/          # 框架层
│   ├── integration-test/   # 集成测试
│   └── playground/         # 开发调试环境
├── packages/               # 公共包
│   ├── common/            # 通用工具包
│   └── frontend/          # 前端应用包
├── tools/                 # 构建工具链
├── docs/                  # 项目文档
└── scripts/               # 构建和部署脚本
```

## 工作空间配置

项目使用 Yarn workspaces 管理多个子包：

- `blocksuite/affine/*` - Affine 核心功能
- `blocksuite/affine/blocks/*` - 块组件
- `blocksuite/affine/fragments/*` - 片段组件
- `blocksuite/affine/gfx/*` - 图形处理
- `blocksuite/affine/inlines/*` - 内联元素
- `blocksuite/affine/widgets/*` - 小部件
- `blocksuite/framework/*` - 框架组件
- `packages/common/*` - 通用包
- `packages/frontend/*` - 前端应用
- `tools/*` - 工具包

## 开发脚本

- `yarn dev` - 启动开发服务器
- `yarn build` - 构建生产版本
- `yarn test` - 运行测试
- `yarn lint` - 代码检查
- `yarn typecheck` - 类型检查

## 部署方式

- `yarn build:standalone` - 构建独立版本
- `yarn build:full` - 完整构建
- `yarn package` - 打包应用

## 项目特点

1. **模块化架构**: 采用 monorepo 结构，模块职责清晰
2. **类型安全**: 全面使用 TypeScript
3. **代码质量**: 完善的 lint 和格式化规则
4. **测试覆盖**: 集成测试和单元测试
5. **多平台支持**: 支持 Web、Desktop、Mobile 等多平台
6. **国际化**: 支持多语言
