# 布局结构与区域职责（功能小节）

> 关联上级：`工作空间/入口与布局.md`

## 1. 布局总览

- 左侧：导航栏（页面树、收藏、标签、集合、论坛入口等）。
- 顶部：工作空间信息、搜索框、用户入口、设置入口。
- 中部：主内容区（文档、白板、全局列表、论坛等）。
- 可选右侧：属性面板或详情面板。

## 2. 左侧导航栏

- 页面树区域；
- 收藏与最近访问区域；
- 标签/集合入口；
- 论坛/社区入口。

## 3. 顶部栏

- 当前工作空间名称与切换入口；
- 全局搜索入口；
- 用户头像与账号菜单；
- 设置入口。

## 4. 主内容区

- 按路由渲染对应页面组件；
- 支持在同一布局下切换不同内容类型。
