# 页面与内容编辑（功能大纲）

> 关联上级：`文档/功能文档/workspace.md` 第 3 章。

- 页面类型
  - 普通文档页面：富文本 + Block 编辑
  - 白板页面：edgeless 画布、图形和连线
  - 数据库页面：表格/看板/画廊等视图
- Block 编辑能力（概览）
  - 文本类：标题、段落、列表、引用、Callout 等
  - 富媒体类：图片、附件、嵌入网页/文档等
  - 白板/图形类：形状、流程图、drawio、excalidraw 等
  - 数据类：数据库块与数据视图
  - 高级表达：代码块、Mermaid、LaTeX 等
- 页面级功能
  - 页面属性：标题、图标、封面、标签、集合、作者信息
  - 页面操作：新建、复制、移动、删除（回收站）
  - 历史与统计：版本历史、浏览/编辑统计
