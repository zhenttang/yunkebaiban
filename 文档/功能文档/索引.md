# 功能文档索引（Features Index）

> 本目录用于从「功能视角」拆分项目文档。每个子文档专注于一块业务或用户功能区域，包含：功能列表、主要交互流程、关键页面和入口。

- `Web端概览.md`：Web 桌面端整体功能（首页、下载、工作空间等）
- `workspace.md`：工作空间系统（文档、白板、集合、标签、附件、分享）
  - `工作空间/入口与布局.md`：工作空间入口、布局结构、选择器
  - `工作空间/页面与内容编辑.md`：页面类型、块编辑、页面属性
  - `工作空间/页面组织与导航.md`：页面树、收藏、标签与集合导航
  - `工作空间/全局视图与列表.md`：All Pages / All Tags / All Collections
  - `工作空间/附件与资源管理.md`：附件列表、预览与操作
  - `工作空间/分享与访问控制.md`：分享链接、访问级别、协同编辑
  - `工作空间/设置与成员管理.md`：工作空间设置、成员与角色、配额
- `forum.md`：论坛与社区功能
  - `论坛/论坛概览.md`：论坛整体功能与角色划分
  - `论坛/浏览与导航.md`：论坛首页、板块浏览、帖子详情
  - `论坛/帖子创作与管理.md`：发帖、草稿、编辑历史
  - `论坛/标签与搜索.md`：标签浏览、搜索结果
  - `论坛/用户与收藏.md`：用户主页、收藏列表
  - `论坛/通知与管理.md`：通知中心、版主面板与管理操作
- `admin.md`：管理后台所有功能（初始化、账号管理、AI 配置、监控、设置等）
  - `管理后台/初始化与安装向导.md`：首次安装、基础配置与服务器初始化
  - `管理后台/管理员认证与访问控制.md`：管理员登录、权限校验、路由保护
  - `管理后台/账号与用户管理.md`：用户列表、搜索、创建、禁用等
  - `管理后台/AI配置中心.md`：模型配置、Key 管理、配额与策略
  - `管理后台/系统监控.md`：运行状态、日志、指标与告警
  - `管理后台/系统设置与模块配置.md`：各模块设置页（存储、邮件、认证、License 等）
  - `管理后台/关于与系统信息.md`：版本信息、自建/云端模式说明
- `ai.md`：AI 功能（AI 块、AI 面板、Copilot、Slides 等）
  - `AI功能/AI概览.md`：AI 能力整体说明与使用场景
  - `AI功能/聊天与侧边面板.md`：AI 聊天面板、上下文与消息流
  - `AI功能/文档内AI与块能力.md`：AI 聊天块、转写块、重写/续写等
  - `AI功能/白板与 Edgeless Copilot.md`：白板 AI 辅助、自动布局、思维导图
  - `AI功能/Slides 与结构化生成.md`：演示文稿生成、结构化内容生成
- `templates-and-clipper.md`：模板中心与剪藏集成功能
  - `模板与剪藏/模板中心.md`：模板列表、预览、导入与应用
  - `模板与剪藏/剪藏集成.md`：剪藏入口、授权、内容落地

> 说明：目前大部分文件会先以大纲形式存在，后续在使用过程中逐步补充细节。
